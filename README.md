# project

krenovator sharing session.

This is an example project, that uses a package `library`, hosted from a Git repo: https://github.com/irfancoder/library

## Instructions

1. Clone repo: 
```bash
  git clone git@github.com:irfancoder/project.git
```
2. yarn install
```bash
  yarn
```

3. start dev server
```bash
  yarn dev
```
